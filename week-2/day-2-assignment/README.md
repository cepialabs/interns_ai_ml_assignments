## Project Description  
Dataset: Email marketing campaign  

This project focuses on analyzing an email marketing campaign using probability concepts.  
The objective of this assignment is to calculate:  
- Probability that a user clicks an email  
- Probability that a user converts given they clicked  
- Conditional probability tables and visualization  

The analysis helps in understanding customer behavior and measuring the effectiveness of email campaigns using data analytics techniques.

## Steps Performed  

1. Imported required Python libraries.  
2. Loaded the email marketing dataset.  
3. Calculated the probability of users clicking the email.  
4. Calculated the conditional probability of conversion given click.  
5. Created conditional probability tables using crosstab.  
6. Visualized results using bar chart and pie chart.  
7. Interpreted the results and derived insights.
