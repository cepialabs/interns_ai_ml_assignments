# Marketing Campaign Data Analysis

## Project Overview
This project analyzes a marketing campaign dataset to understand customer spending
behavior and engagement patterns. The dataset includes customer spend, email engagement,
ad clicks, and website visits.

## Dataset
- File: `data/marketing_campaign.csv`
- Records: 60 customers
- Features:
  - customer_id
  - spend_usd
  - email_opens
  - ad_clicks
  - website_visits

## Analysis Objectives
1. Calculate average spending per customer
2. Identify the top 10% of spenders
3. Detect skewness in customer engagement metrics

## Key Results
- Average customer spending: $227.53
- Top 10% spender threshold: $392.25
- Website visits show moderate right skew, indicating high activity from a subset of users

## Tools Used
- Python
- Pandas
- NumPy

## Use Cases
- Customer segmentation
- Targeted marketing strategies
- Campaign performance evaluation
