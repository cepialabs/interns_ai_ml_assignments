# House Prices Prediction Project

## 📌 Objective

Train a Random Forest model to predict house prices and compare its R²
score with Linear Regression.

## 📊 Dataset

-   File: house_prices_dataset.csv
-   Target variable: price

## 🚀 Models Used

-   Linear Regression (Baseline)
-   Random Forest Regressor

## 📈 Evaluation Metric

-   R² Score

## 📊 Visualization

-   Feature importance plot from Random Forest

## ▶️ How to Run

1.  Open the notebook: house_prices_random_forest.ipynb
2.  Run all cells
3.  Compare R² scores
4.  View feature importance plot
