# Machine Learning Models Project

## Overview

This project demonstrates: 1. Housing price prediction using Linear
Regression. 2. Email spam classification using Logistic Regression.

## Files

-   housing_prices.csv : Synthetic housing dataset.
-   email_spam.csv : Email spam dataset.
-   ml_models.ipynb : Jupyter notebook with implementation.

## Requirements

-   pandas
-   scikit-learn
-   numpy
-   jupyter

## How to Run

1.  Install required libraries.

2.  Open the Jupyter notebook:

        jupyter notebook ml_models.ipynb

3.  Run all cells.
