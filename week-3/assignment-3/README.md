
# KNN and Decision Tree ML Project

## Files Included
- spam_email_dataset.csv : Sample dataset for spam email classification
- ml_knn_decision_tree.ipynb : Jupyter Notebook with KNN & Decision Tree models
- README.md : Project overview

## Models
- KNN Classifier: Classifies emails as spam or not spam
- Decision Tree Classifier: Spam classification
- Decision Tree Regressor: House price prediction using California Housing dataset

## How to Run
1. Place all files in the same directory
2. Open the notebook
3. Run cells sequentially
