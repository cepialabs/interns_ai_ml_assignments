
# House Prices Feature Engineering Project

## Files
- house_prices.csv : Sample housing dataset
- house_prices_feature_engineering.ipynb : Feature engineering and feature importance analysis
- README.md : Project overview

## Steps Performed
1. Created house age from year built
2. Created price per square foot feature
3. Encoded categorical features (Neighborhood, Overall Condition)
4. Trained Random Forest model to analyze feature importance

## Requirements
- pandas
- scikit-learn
- matplotlib
