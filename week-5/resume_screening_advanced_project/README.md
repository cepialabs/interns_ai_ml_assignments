# Resume Screening ML Model

## 📌 Business Problem

Manual resume screening is time-consuming and inefficient.
This project automates resume filtering using Machine Learning
and NLP techniques.

## 🎯 Objective

Build an NLP-based model that:
- Classifies resumes into job roles
- Ranks resumes based on job description relevance

## 📊 Dataset

- 250 Resume samples
- 5 Job Roles:
  - Data Scientist
  - Web Developer
  - Android Developer
  - DevOps Engineer
  - UI/UX Designer

Dataset Fields:
- Resume (Text)
- Job_Role (Label)

## 🛠 Technologies Used

- Python
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
- Pandas
- Matplotlib

## ⚙ Model Workflow

1. Text Preprocessing
2. TF-IDF Feature Extraction
3. Label Encoding
4. Logistic Regression Model
5. Model Evaluation

## 📈 Model Performance

- Accuracy: ~85% to 95% (depending on random split)

## 🚀 How to Run

1. Clone the repository
2. Install requirements:

pip install -r requirements.txt

3. Run the notebook
