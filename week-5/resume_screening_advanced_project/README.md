🔹 Project Summary

This project builds an AI-based resume screening system that automates candidate classification and ranking based on job requirements using Natural Language Processing and Machine Learning.

🔹 Business Problem
HR departments manually review hundreds of resumes, which is:
Time-consuming
Error-prone
Biased
This system automates screening and ranks candidates based on job description similarity.

🔹 Technical Stack
Python
Scikit-learn
TF-IDF Vectorization
Logistic Regression
Cosine Similarity
Seaborn & Matplotlib

🔹 Key Features

✔ Resume Classification
✔ Candidate Ranking
✔ Skill Extraction
