# Exploratory Data Analysis Project

## Objective
To analyze the dataset and extract meaningful insights using EDA techniques.

## Dataset
Student Performance Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
- Data loading and understanding
- Data cleaning
- Exploratory data analysis
- Visualization
- Insights generation

## Key Insights
1.Students with higher attendance tend to score better in exams.

2.Female students perform slightly better in reading and writing.

3.Math scores show greater variation compared to other subjects.

4.Attendance has a moderate positive correlation with overall performance.

5.Few outliers indicate students with exceptionally high or low scores.

## Conclusion
EDA helped identify patterns and relationships in the dataset.
