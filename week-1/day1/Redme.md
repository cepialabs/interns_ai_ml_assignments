# Day 2 Assignment – House Price Analysis

## Objective
Generate 10,000 random house prices using NumPy and perform basic data analysis.

## Tasks Performed
1. Generated 10,000 random house prices using NumPy
2. Calculated the average house price
3. Removed outliers using the IQR (Interquartile Range) method
4. Visualized the price distribution using a histogram

## Tools & Libraries Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## File Structure
- House_Price_Assignment.ipynb ? Main Jupyter Notebook
- README.md ? Assignment documentation

## Output
- Average house price printed in notebook
- Cleaned price distribution visualized after removing outliers
