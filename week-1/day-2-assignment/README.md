# Netflix Titles Dataset Analysis
Date:20 January 2026

## Project Description
This project is an exploratory data analysis (EDA) of the Netflix Titles dataset. The objective of this analysis is to understand the distribution of content available on Netflix by genre, content type, year of addition, and country of production. The analysis is performed as part of an internship assignment using Python.

## Dataset Information
- **Dataset Name:** Netflix Titles Dataset  
- **Source:** Kaggle  
- **Dataset URL:** https://www.kaggle.com/datasets/mahmoudtaya/netflix-titles  

The dataset contains information about movies and TV shows available on Netflix, including details such as title type, genre, country, date added, and release year.

## Objectives
The main objectives of this project are:
- Identify the top 10 most frequent genres on Netflix
- Compare the distribution of Movies and TV Shows
- Analyze the trend of content added to Netflix over the years
- Identify the country producing the most Netflix content

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git and GitHub

## Methodology
- Loaded the dataset using Pandas
- Cleaned and processed columns such as `listed_in`, `country`, and `date_added`
- Extracted year-wise information from the date column
- Performed aggregation and value counts for analysis
- Visualized insights using bar charts and line plots

## Key Observations
- Movies are more prevalent than TV Shows on Netflix
- Drama and International genres appear most frequently
- Content addition increased significantly after 2016
- The United States is the largest contributor to Netflix content
