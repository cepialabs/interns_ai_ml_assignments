## Assignment Description  
This assignment focuses on exploratory data analysis (EDA) using the Student Performance Dataset.  
The aim is to analyze student scores and understand how factors like gender and attendance impact performance.

## Dataset  
Name: Student Performance Dataset  
Source: Kaggle (UCI Student Performance Dataset)  
URL: https://www.kaggle.com/datasets/devansodariya/student-performance-data
The dataset contains:
- Demographic information
- Family background
- Study habits
- Attendance details
- Academic scores (G1, G2, G3)

Total Records: 395  
Total Columns: 33  

## Objective  
To analyze student performance using visualization techniques and identify meaningful patterns.

## Tasks  
1. Score distribution plot  
2. Gender vs performance chart  
3. Correlation heatmap  
4. Attendance vs score trend chart  

## Tools & Libraries  
- Python  
- Pandas  
- Matplotlib  
- Seaborn
## Key Insights  
- Students with fewer absences tend to have higher final scores.  
- Female students show slightly better average performance.  
- G1 and G2 are strongly correlated with G3.
