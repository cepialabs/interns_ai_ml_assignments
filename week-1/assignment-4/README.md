## Overview
This project analyzes student performance data to understand score distribution, gender-based performance, correlations between variables, and the relationship between attendance and scores.

## Dataset
Student Performance Dataset containing:
- Gender
- Attendance
- Math, Reading, and Writing scores

## Objectives
- Visualize score distribution
- Compare performance by gender
- Analyze correlation between academic factors
- Study attendance vs score trends

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
1. Score distribution histogram
2. Gender vs performance bar chart
3. Correlation heatmap
4. Attendance vs score scatter plot
